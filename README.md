# Emotion Detection from Text

This project performs multi-class emotion classification, identifying the emotion expressed in a sentence.

Unlike sentiment analysis, this task predicts specific emotional states rather than general polarity.

---

## Emotions Classified
- Joy
- Sadness
- Anger
- Fear
- Love
- Surprise
